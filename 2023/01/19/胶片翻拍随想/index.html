胶片翻拍随想

Posted on 2023-01-19 Edited on 2023-09-25 In Film Disqus:

基于实际操作的一些关于翻拍胶片的想法。

数码相机和胶片的准备

关闭机内暗角抑制。

保持胶片的平整和干净。

尽可能保持胶片和相机焦平面平行。如果胶片有些卷起来，最后翻拍的结果会有一种类似于移轴摄影的效果。

相机内色域设置为Adobe RGB。避免使用sRGB。

准确对焦。

相机切换至手动模式，锁定快门速度和光圈大小。

打开相机直方图显示，防止高光溢出。

相机可以外接显示器。

尽量拍摄RAW文件，而不是JPEG文件。

使用快门线。

相机色温设定尽可能贴近光源色温，减少后期工作量。

翻拍同一张底片时，对焦区域尽可能选择底片不同的区域。

关于放大倍率

胶片冲洗后建议单张底片翻拍数量不少于4张，以便后期在Photoshop中进行堆栈操作时挖掘出更多的底片细节。

翻拍时可以使用微距镜头，或者普通镜头加上微距环。普通镜头接上微距环后的最大放大倍率可用以下公式计算：

加微距环后最大放大倍率 = 镜头本身最大放大倍率 + 转接环长度/镜头焦距

例：

尼康Z口50mm f/1.8 S本身最大放大倍率为0.15倍。即物体在传感器上成像面积最大为物体实际面积的0.15倍。

另购得转接环两具，合并延伸长度为36毫米。

带入公式计算可知，加微距环后最大放大倍率为 0.15 + 36/50 = 0.87。

注意，接上微距环后镜头的最近对焦距离会缩短，并且可能无法在无穷远处合焦。换言之，镜头可以合焦的距离发生了变化，不再是原来的最近对焦距离一直到无穷远。如果是专用的微距镜头，则可以切换微距和非微距的工作模式，从而在不同的放大倍率上都能精准合焦。

普通镜头加微距环并不是唯一的提高镜头放大倍率的方法。镜头放大倍率也不是在微距摄影中在镜头的选择上唯一需要考虑的因素。

Photoshop流程

  1. File -> Scripts -> Load Files into Stacks.
  2. Browser and select files.
  3. Select “Attempt to Automatically Align Source Images” and “Create Smart Object after Loading Layers”. Click OK.
  4. Filter -> Camera Raw Filter: adjust color temperature, sharpening, noise reduction and color noise reduction. Click OK.
  5. Layers -> Smart Objects -> Stack Mode -> Mean.
  6. Image -> Image Size.可以调整分辨率，挖掘更多底片信息。
  7. Crop the image.
  8. File -> Export. 建议同时保存JPEG和PNG两种格式。JPEG可以生成更小体积的文件，方便放在手机里自己看或者用于网络传播。以PNG格式导出的文件体积更大，可以保存更多色彩信息，可以用于高质量打印。

其中Camera Raw Filter、调整照片尺寸和选择堆栈模式可以在Photoshop中记录三个action，减少重复劳动。

后记

多张翻拍堆栈确实可以获得更多细节，但是需要更加耐心的后期。

扫描仪固然好，但是扫描仪只能扫描，不能拿出去拍照。镜头可以拍照也可以翻拍胶片。

目前只尝试了反转片和黑白负片。因为彩色负片在去色罩上的不确定性目前还没有尝试。

我第一次进行翻拍的时候，因为机内暗角抑制没有关闭，在尼康的免费软件NX Studio里校色的时候经常出现牛顿环。关闭机内暗角抑制后在Photoshop Camera Raw里校色的时候没有再出现牛顿环。目前无法确定是否解RAW文件用的软件的问题还是相机设置问题。

胶片也是有上限的。过量堆栈无法无限量地挖掘细节。

用胶片拍了什么才是最重要的。
